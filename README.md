## 🚀 Getting Started
To run the app locally:

1. **Clone the repo**
```bash
git clone https://github.com/Niraj1903/Todolist.git
cd Todolist
````

2. **Install dependencies**

```bash
npm install
```

3. **Run the dev server**

```bash
npm start
```

4. Open `http://localhost:3000` to view the app.

## ⚙️ Build for Production

```bash
npm run build
```

This creates an optimized `build/` folder you can deploy to Netlify, Vercel, etc.
